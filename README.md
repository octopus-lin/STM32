基于STM32F10x微控制器设计的智能台灯控制系统，集成多维度环境感知模块、PWM智能调光技术、语音交互与云端远程控制功能。系统通过光照传感器、红外人体传感器、超声波测距模块实时监测环境，实现亮度自适应调节、色温切换、护眼报警与定时休息等功能，构建"感知-决策-控制-反馈"的闭环系统，为用户提供低成本、高实用性的护眼照明解决方案。

多传感器融合：集成光敏传感器检测环境光照强度、红外传感器识别人体存在、超声波传感器测量距离、MPU6050检测台灯姿态，通过多传感器数据融合实现智能环境感知。

智能调光算法：根据环境光照强度自动调节WS2812B RGB LED亮度，实现三档亮度调节（暗环境高亮度、中等环境中等亮度、亮环境低亮度），结合人体检测实现人来灯亮、人走灯灭的节能模式。

云端远程控制：通过ESP8266 WiFi模块接入机智云物联网平台，实现手机APP远程控制台灯开关、亮度调节、定时设置等功能，支持数据上报和云端存储。

超声波测距报警：使用HC-SR04超声波传感器实时监测用户与台灯距离，当距离小于设定阈值时触发蜂鸣器报警，并通过串口发送语音提醒，保护用户视力。

PWM调光控制：通过定时器PWM输出控制WS2812B RGB LED，实现亮度无级调节和颜色切换，采用DMA传输提高控制效率。

OLED人机交互：使用0.96寸OLED显示屏实时显示时间、光照强度、超声波距离、当前档位等信息，支持按键切换自动/手动模式，提供直观的用户界面。

RTC时钟管理：集成RTC实时时钟模块，实现定时开关灯功能，支持设置特定时间自动开启或关闭台灯，满足用户个性化需求。
